# Подробное описание класса: Junction

________________________________________
## 1. Общая информация
•	**Имя класса:** Junction  
•	**Тип:** Activity  
•	**Назначение:** Это переходный экран ("развязка"), который появляется после входа или регистрации. Его главная особенность — визуальный эффект движущейся тени у текста.  
•	**Взаимодействие:** Принимает данные пользователя через `Intent`. Управляет выпадающим меню через `BaseMenuClass`.

________________________________________
## 2. Переменные (Поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `tv1` | `TextView` | Текстовое поле с эффектом. | В методе `goShadow`. |
| `totalTime` | `long` | Длительность эффекта (10 сек). | В таймере. |
| `radius` | `long` | Радиус движения тени. | В математических расчетах тени. |
| `ivPUMJ` | `ImageView` | Кнопка вызова меню. | В `onCreate`. |
| `baseMenuClass` | `BaseMenuClass` | Управление меню и голосом. | Для показа меню. |

________________________________________
## 3. Методы класса
### Имя метода: onCreate
•	**Что делает:** Устанавливает разметку, находит элементы, инициализирует меню и запускает метод `goShadow`.

### Имя метода: goShadow
•	**Тип:** private  
•	**Что делает:**  
1. Запускает таймер `CountDownTimer`.  
2. На каждом "тике" (каждые 100 мс) вычисляет новые координаты `dx` и `dy` для тени по формуле круга (`cos` и `sin`).  
3. Применяет метод `setShadowLayer` к тексту, чтобы тень плавно вращалась вокруг него.  
4. По завершении убирает тень.  
•	**Важно понимать:** Здесь используется тригонометрия для создания плавного кругового движения.

________________________________________
## 4. Жизненный цикл
•	**onCreate()**: Запуск анимации и меню.  
•	**onDestroy()**: Остановка TTS.

________________________________________
## 5. Взаимодействие с интерфейсом (UI)
•	**TextView (tv1)**: Демонстрация работы с программным изменением теней.  
•	**ImageView (ivPUMJ)**: Кнопка для доступа к настройкам и навигации.

________________________________________
## 6. Взаимодействие с другими компонентами
•	**BaseMenuClass**: Используется для стандартных действий (Назад, Выход).

________________________________________
## 7. Общая логика класса
При входе на этот экран пользователь видит текст, тень которого начинает плавно бегать по кругу в течение 10 секунд. Это служит индикатором того, что переход выполнен успешно.

________________________________________
## 8. Упрощенное объяснение
**Объяснение простыми словами:**  
`Junction` — это **зал ожидания** с красивым спецэффектом. Когда вы заходите, текст на экране "оживает": его тень начинает кружиться, как будто вокруг текста бегает фонарик. Это сделано для того, чтобы приложение выглядело более динамичным.
