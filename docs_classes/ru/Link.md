# Подробное описание класса: Link

________________________________________
## 1. Общая информация
•	**Имя класса:** Link  
•	**Тип:** Обычный класс (Модель данных)  
•	**Назначение:** Этот класс представляет собой модель "Сылки". Он используется для хранения информации об интернет-ресурсе или видео, которое будет отображаться в списке.  
•	**Взаимодействие:** Объекты этого класса создаются в `LinksFragment` при чтении данных из базы данных.

________________________________________
## 2. Переменные (Поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `linkName` | `String` | Название ссылки. | Для отображения в списке. |
| `linkDescription` | `String` | Описание ресурса. | В конструкторе и методах доступа. |
| `linkType` | `String` | Тип ссылки ("video" или "web"). | Для выбора способа открытия (плеер или браузер). |
| `linkID` | `String` | URL адрес или ID видео на YouTube. | Для загрузки контента. |

________________________________________
## 3. Методы класса
### Конструктор Link(...)
•	**Тип:** public  
•	**Параметры:** `linkName`, `linkDescription`, `linkType`, `linkID`.  
•	**Что делает:** Создает объект ссылки с заполненными данными.

### Геттеры и Сеттеры (например, `getLinkName`, `setLinkName`)
•	**Тип:** public  
•	**Что делают:** Позволяют получать и изменять значения полей.

### Метод toString()
•	**Тип:** public  
•	**Что делает:** Возвращает текстовую строку с данными объекта. Используется для отладки.

________________________________________
## 4. Жизненный цикл
Не применимо.

________________________________________
## 5. Взаимодействие с интерфейсом (UI)
Напрямую не взаимодействует. Данные из объекта `Link` используются `LinksFragment` для заполнения списка `ListView`.

________________________________________
## 6. Взаимодействие с другими компонентами
Используется вместе с `HelperDB` (откуда берутся данные) и `CustomAlertDialog` (куда передается `linkID` для показа).

________________________________________
## 7. Общая логика класса
Это простой "контейнер" для хранения информации об одной ссылке.

________________________________________
## 8. Упрощенное объяснение
**Объяснение простыми словами:**  
`Link` — это **карточка товара** в интернет-магазине. На карточке написано название, что это за товар (видео или сайт) и ссылка на него. Сама карточка ничего не делает, она просто хранит информацию, чтобы её можно было прочитать и показать покупателю.
